# rest-magpie (planning workspace)

This is the **planning** workspace for `rest-magpie` — a Model Context Protocol server for REST API calls with schema-first responses.

The implementation hasn't started yet. This folder contains:

- `docs/specs/` — the design spec (the contract)
- `docs/plans/` — the implementation plan (TDD task list for the implementing agent)
- `AGENTS.md` — orientation for the agent that will implement the plan

When ready to implement, run an agent with the `superpowers:subagent-driven-development` skill against the plan in `docs/plans/`.

The actual project (`package.json`, `src/`, tests, Docker, etc.) will be created by Task 1 of the plan.
